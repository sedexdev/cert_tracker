Certification Tracking Space

About

This is a project I am working on to help me track my progress through various certifications. I have a habit of starting a certification and then getting distracted by another one, or just not finishing the one I started. This project is an attempt to help me stay focused and organised. I have opted to make this public in the hopes that it will help others as well.

Features

This application wil allow you to:

  • Track separate certifications in dedicated instances
  • Upload course links and add sections sections
  • Include links to articles and documentation
  • Track flash card creation
  • Set exam dates
  • Tag certifications for quick searching

In-progress features include:

  • Timeline of achievements
  • Email reminders to help stay on track
  • Statistical analysis dashboard
  • In-app notes and directory structure
  • Digital flash cards
  • Cost analysis
  • Practice test resource organiser
